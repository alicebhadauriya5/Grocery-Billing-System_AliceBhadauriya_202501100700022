## Problem Statement

A grocery store wants to calculate the total cost of items purchased by a customer.

The program should:

- Accept price input for 3 different items
- Calculate the total cost
- Apply a 10% discount if the total exceeds $50
- Display the original total, discount, and final payable amount

---

## Approach

1. Take input prices for three items from the user.
2. Add the prices to calculate the total amount.
3. Check if the total is greater than $50.
4. If yes, apply a 10% discount.
5. Display the original total, discount, and final payable amount.
